# FarmApp
Android Application developed using Java and Flutter to monitor sensor values of a farmland and automatically start/stop irrigation. <br>
<b>1) - Temperature </b> <br>
<b>2) - Humidity </b> <br>
<b>3) - Soil Moisture </b> <br>
